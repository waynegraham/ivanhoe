What is Ivanhoe?

Ivanhoe is a literary game where players engage a version of a text and modify it, thereby creating a new interpretive instance of the work.  Each player may choose to react to other interpretations and locate his or her interpretation within the universe of interpretive possibility.  In the context of the game, the universe of possibility is simply all interpretations by other players.  In the large scheme of things, an Ivanhoe game exists as a constellation among infinite possible interrelated interpretations.

Typical gameplay involves modifying the Declared Edition of the text, adding new documents, online chat with other players and summarizing one's move in the forum.  A move is composed of one or more actions, such as addition or deletion.  When finished, the move should, be submitted for other players to see.

How to use this Help

The navigator bar on the left provides a table of contents, an index and a full text search engine.

  • Use the table of contents if you are using Ivanhoe for the first time and want to get an overview of its features as well as to get familar with carrying out basic tasks. The table of contents is in a logical order.
  • Use the index, if you search for a task that can be described with a certain keyword. Hopefully the index contains the word and takes you quickly to the underlying helpfile. It is in alphabetical order.
  • Use the full text search engine, if you were not successful with the index. The results of the search are ranked after the number of matches into 5 categories.
