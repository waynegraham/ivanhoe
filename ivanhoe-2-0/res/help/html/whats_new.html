What's New?

The following are new changes for milestone 4 (July 28, 2004):

  • Completed visualization for player centric view of the game
  • Chat moved out of tab and into a floating window
  • User configurable color controls for visualization
  • Ability to delete a document you added before submitting your move
  • Ability to create links between documents
  • Ability to create a new document (import of html or txt still available)
  • Move summary shown prior to publishing the move
  • Timeline control is complete, with playback and next/prev step buttons
  • Timeline control shows proportional tick marks to indicate time elapsed between moves
  • Server now shows a list of games available
  • Player can create a user account
  • Player can create a new game
  • Notifcation of moves and game activity sent to players via email
  • Ability to create new documents from an image
  • Ability to insert images directly into documents

The following are new changes for milestone 3:

  • Initial visualization for player discourse field
  • User interface elements placed into trays on left side to maximize space for visualization and document editing
  • Main menu moved to right side, appears upon mouseover
  • Document editor has indicator in the "gutter" to navigate through actions. Gutter has color indicators for actions within the current move.
  • Next and Prev arrow buttons to cycle through gutter nibs
  • New buttons to remove single action or all actions in editor
  • Overriden look and feel for internal windows
  • New button in toolbar to remove all actions, which is useful when changing your mind about adding a document as a move
  • Players can now comment/annotate in each other's discourse field

The following are new changes for milestone 2:

  • Search is greatly enhanced to search all documents
  • Cut and paste is now implemented
  • Hovering pointer over actions shows mini summary of action
  • Move Inspector makes it possible to view the documentary state of another's move at various points in time
  • Numerous bug fixes to the editor environment
  • General UI enhancements (new icons, etc)
  • Delete actions can be hidden from view
  • Discussion (forum) enables responding directly to another player's entry
  • All game state is stored on the server
  • Added a page break indicator in the editor

  • Many changes on the server side
  • Documents are loaded on demand from the server (scalability)
